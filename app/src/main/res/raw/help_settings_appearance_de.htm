• Hintergrund

    Mit diesem Schalter steuern Sie, ob ein heller oder dunkler Hinter­grund verwendet werden soll.
    Bei automatischer Auswahl wird tagsüber ein heller Hinter­grund gewählt, sonst ein dunkler Hinter­grund.

  • Ausrichtung

    Diese Einstellung kontrolliert die Bild­schirm­ausrichtung. Normaler­weise orientiert sich der Bild­schirm­inhalt anhand der Gerätesen­soren. Die Ausrichtung kann aber auch vertikal oder horizontal fixiert werden.
    Diese Einstellung wird nicht auf Videos angewendet.

  • Zeitdarstellung

    Dieser Schalter beeinflusst, wie das Datum des Artikels dargestellt wird. Bei Auswahl von „Absolut” wird das Datum im Format „01.11.2011 11:11” angezeigt. Wenn „Relativ” ausgewählt ist, dann wird der verstrichene Zeit­raum an­ge­geben, beispiels­weise „vor 2 Stunden”.

  • Anführungszeichen

    Mit diesem Schalter steuern Sie, ob falsche (" ") Anführungs­zeichen korrigiert werden sollen. Weitere Informa­tionen zu Anführungs­zeichen im Deutschen auch in der Wikipedia.

  • Laufschrift in Titelzeile

    Der Inhalt der obersten Zeile kann als Lauf­schrift darge­stellt werden, wenn er nicht voll­ständig auf den Bildschirm passt.

  • Schriftvergrößerung

    Der Schieberegler steuert die Schrift­größe in der Nach­richten­liste.

  • Schrift importieren

    Sie können eine Schrift impor­tie­ren, die auf dem Gerät als Datei im TTF-Format vor­liegt. Nach er­folg­reichem Import wird die Schrift für den über­wiegen­den Teil des Textes ver­wen­det.

  • Importierte Schrift löschen

    Wenn Sie eine Schrift impor­tiert ha­ben, können Sie diese hier wie­der löschen. Dadurch wird Speicher­platz frei­gegeben und die ursprüngliche Dar­stellung wieder­hergestellt.
