Wenn Sie Nach­richten mit be­stimm­ten In­halten nicht lesen möchten, können Sie hier Filter de­finieren, die diese Nach­richten­inhalte ver­bergen.
Um eine Nach­richt zu ver­bergen, muss der ange­gebene Begriff irgendwo in einem Wort bzw. am Anfang oder am Ende eines Wortes er­scheinen.
Ein Filter mit dem Inhalt „musk” würde beispiels­weise keine Nach­richten zu­lassen, die das Wort „Muskateller” enthalten, es sei denn, die Option „am Ende” ist gewählt.

Groß- und Klein­schreibung wird ignoriert.

Optional können die Filter auch auf die Kategorien angewendet werden. So kann z.B. durch einen Filter mit dem Inhalt „sport” die Kategorie „Sport” aus­geblendet werden.
